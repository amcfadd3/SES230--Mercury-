# Figures

This folder contains charts, maps, screenshots, and report images used in the final project.

- `report_image_1.png` - CA_EQ_Shallow attribute table figure used in the Data section
- `report_image_2.png` - Python workflow flow chart used in the Code / Methods section
- `report_image_3.png` - California earthquake and rock type map used in the Code / Methods section
- `report_image_4.png` - example GeoPandas code and output used in the Code / Methods section
- `report_image_5.png` - earthquake distribution by rock type pie chart used in the Results section
- `CA_EQ_Shallow_*.png` - attribute analysis figures generated from the CA_EQ_Shallow dataset
- `California_Surface_Rock_Type_Percentages.png` - surface rock type percentage visualization
- `Cali_EQ_Count_Chart.svg` - earthquake count chart
- `attribute_table.png` and `earthquake_pie_chart.png` - additional project visual outputs

Figures were generated from the project notebooks, Python scripts, GIS outputs, or final report exports.
