# Earthquake Location Rock Type Correlation

Repository structure modeled after the recommended SES230 group project layout.

## Project folders

- `code/` - analysis and utility scripts
- `data/raw/` - raw source data files
- `data/processed/` - processed datasets ready for analysis
- `figures/` - charts, maps, and display graphics
- `docs/` - supporting documentation
- `zotero/` - bibliography and reference library
